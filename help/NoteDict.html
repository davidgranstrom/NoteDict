NoteDict a dictionary for pitch notations


Inherits from: Object


Allows easy access to various notations when working with pitches.


See also: Object 


Creation / Class Methods


*new (style, output)

Creates a new IdentityDictionary containing note names.

style available styles are: \traditional, \helmholtz and \MIDI (default: \traditional)

output output in midi note numbers or Herz (default: \MIDI)

// create a new dictionary

n= NoteDict();


n['c1']; // get one note

n[[ 'c', 'e', 'g' ]]; // get an array of notes

*numberToNote (num, style)

Not intended to be used directly. 

Extends SimpleNumber instead, see below.

num a MIDI note number

style in which style to display the output

// ask MIDI notes for their pitch name

60.asNote;

60.asNote(\MIDI);

// works on arrays too

[ 60, 64, 67 ].asNote;

NOTE: when using style \helmholtz the octave symbol will 

not be displayed correctly in the post window.


Examples


'middle c' in all notation styles


'c1' -> \traditional (german)

'c´' -> \helmholtz

'C4' -> \MIDI

Both sharp '#' and flat 'b' notation is supported.

The ranges are the following:

\traditional =  C3 ->  g6

\helmholtz =  C,,, ->  g´´´´´´

\MIDI  =  C-1 ->  G9


//-- sound examples

s.boot;

x= play{|freq| SinOsc.ar(freq) * 0.15 };


// helmholtz notation with frequency output

n= NoteDict(\helmholtz, \freq);


x.set(\freq, n['gb´'].postln);  // one-lined g flat

x.set(\freq, n['c#´´'].postln); // two-lined c sharp


x.free;


(

// sequencing with patterns

// dictionary returns midi notes

s.waitForBoot{ 


// create a new dictionary with default values

n= NoteDict(); 

// make some chords..

~c=  n[[ 'c', 'e', 'g' ]]; 

~am= n[[ 'a', 'c', 'e' ]]; 

~em= n[[ 'e', 'g', 'b' ]];


SynthDef(\arp, {|out=0, freq, cfreq|

var src, env;

src= RLPF.ar(Saw.ar(freq*[0.99,1,1.01]).mean, cfreq*[1,0.98], Rand(0.25, 1));

env= EnvGen.ar(Env.perc, levelScale:0.25, doneAction:2);

Out.ar(out, src*env);

}).add;

    

s.sync;

    

p= Pbind(

\instrument, \arp,

\midinote, Pseq([ 

Pseq(~em, 8) + Pn(Pshuf([ 0, 12, 24 ], 3)), 

Pseq(~c, 12) + Pn(Pshuf([ 0, 12, 24 ], 3)),

Pseq(~am, 4) + Pn(Pshuf([ 0, 12, 24 ], 3)) 

], inf),

\cfreq, Pseg(Pseq(n[[ 'c1', 'c5' ]].midicps, inf), Pseq([ 16, 16 ], inf), \sine),

\dur, 1/8

).play;

}

)

p.stop;
